# REFATORAÇÃO DE APP

## OBJETIVO DO APP

- Servir como uma página inicial que permita buscar no browser, acessar e salvar sites.

## PLANO

- Transformar esse app em uma extensão que carrega esta página toda vez que o usuário abre uma aba (O Brave não permite isso).
- Manter o atual design, layout, cores, fontes, tamanhos, listeners, funcionalidades, mecanicas. E passar para react.

## NOVA STACK

- Vite
- React
- Tailwindcss

- O Agente tem permissões de usar comandos como: bun
